Autonomous indoor service robots require real-time perception and navigation while maintaining low power consumption for extended operation. Conventional robotic platforms commonly rely on GPU-based systems, such as the NVIDIA Jetson Nano, which provide high computational performance but suffer from significant energy demands. This paper addresses the challenge of achieving efficient real-time processing under constrained power budgets by proposing an FPGA-based autonomous service robot architecture.

The proposed system integrates a Nexys A7 FPGA with a Raspberry Pi 4 to perform real-time perception, mapping, and navigation using LiDAR and stereo vision data. A discretized grid map is constructed on the FPGA, enabling efficient execution of the A* pathfinding algorithm and obstacle avoidance in dynamic indoor environments. Classical edge detection techniques are also implemented to enhance real-time environmental awareness.

The novelty of this work lies in the use of FPGA-based parallel processing to offload computationally intensive navigation tasks, an architecture designed to significantly improve energy efficiency compared to traditional GPU-based solutions. Additionally, the system provides a modular hardware–software co-design framework that supports real-time operation with reduced power consumption.

This work demonstrates a practical and scalable approach for autonomous robotics. Crucially, it establishes the methodological and physical foundation for future quantitative comparisons between FPGA and GPU architectures in mobile robotic applications, with implications for healthcare logistics, office automation, and indoor delivery systems.
